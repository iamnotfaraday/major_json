# 中国大学生专业目录树形数据 (JSON)

本项目包含了根据教育部最新发布的专业目录整理而成的树形结构数据，涵盖了本科生及研究生两个层级。

## 1. 本科生专业目录结构 (Undergraduate)

本科专业代码逻辑：

- **一级（学科门类）**：2位代码（如 `01` 哲学）。
- **二级（专业类）**：4位代码（如 `0101` 哲学类）。
- **三级（专业）**：6位代码（如 `010101` 哲学），部分包含后缀 `K` (国家控制)、`T` (特设)、`TK`。
- **特殊处理**：针对 `14 交叉学科` 等跳过二级分类的目录，已自动补全逻辑。

### 本科 JSON 示例

```json
{
    "01-哲学": {
        "0101-哲学类": {
            "code": "0101",
            "sub_majors": [
                "010101-哲学",
                "010102-逻辑学",
                "010103K-宗教学",
                "010104T-伦理学"
            ]
        }
    },
    "08-工学": {
        "0809-计算机类": {
            "code": "0809",
            "sub_majors": [
                "080901-计算机科学与技术",
                "080902-软件工程",
                "080910T-数据科学与大数据技术"
            ]
        }
    },
    "14-交叉学科": {
        "1400-未分类/通用": {
            "code": "1400",
            "sub_majors": [
                "140001TK-未来机器人",
                "140002TK-交叉工程"
            ]
        }
    }
}
```

## 2. 研究生专业目录结构 (Graduate)

研究生专业代码逻辑与本科略有不同，通常采用一级门类 -> 一级学科 (二级) -> 二级学科 (三级) 的模式。

### 研究生 JSON 示例

```json
{
    "01-哲学": {
        "0101-哲学": {
            "code": "0101",
            "sub_majors": [
                "010101-马克思主义哲学",
                "010102-中国哲学",
                "010103-外国哲学",
                "010108-科学技术哲学"
            ]
        }
    },
    "08-工学": {
        "0812-计算机科学与技术": {
            "code": "0812",
            "sub_majors": [
                "081201-计算机系统结构",
                "081202-计算机软件与理论",
                "081203-计算机应用技术"
            ]
        }
    }
}
```

## 3. 数据处理说明

- **来源**：教育部《普通高等学校本科专业目录》及《研究生教育学科专业目录》。
- **解析工具**：Python + `pdfplumber`。
- **清理逻辑**：
  - 去除单元格内的换行符 (`\n`)。
  - 自动截断“注：...”等备注信息，仅保留核心专业名称。
  - 兼容 `TK/T/K` 等特殊后缀代码。
